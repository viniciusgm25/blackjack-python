# Blackjack (21) em Python

Projeto desenvolvido para praticar lógica de programação, organização de código e regras de negócio utilizando Python.

## Funcionalidades
- Simulação do jogo Blackjack (21)
- Sistema de baralho embaralhado
- Regras oficiais do Ás (1 ou 11)
- Turno do jogador e do dealer

## Conceitos aplicados
- Programação orientada a objetos
- Estruturas condicionais
- Manipulação de listas
- Separação de responsabilidades
